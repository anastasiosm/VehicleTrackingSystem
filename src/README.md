# Vehicle Tracking System

Ένα σύστημα διαχείρισης και παρακολούθησης στόλου οχημάτων σε πραγματικό χρόνο, αναπτυγμένο σε περιβάλλον .NET.

## Επισκόπηση Έργου

Το Vehicle Tracking System επιτρέπει την καταγραφή και την ανάλυση των γεωγραφικών δεδομένων (GPS) οχημάτων. Το σύστημα είναι σχεδιασμένο να δέχεται στίγματα θέσης, να τα επικυρώνει βάσει γεωγραφικών περιορισμών και να παρέχει πληροφορίες για τη διαδρομή και την κατάσταση των οχημάτων μέσω ενός REST API.

### Βασικά Χαρακτηριστικά

*   **Καταγραφή GPS:** Υποστήριξη για μεμονωμένες και μαζικές (batch) υποβολές θέσεων οχημάτων.
*   **Γεωγραφικός Περιορισμός (Athens Bounding Box):** Το σύστημα διαθέτει μηχανισμό επικύρωσης που δέχεται συντεταγμένες μόνο εντός των ορίων της Αθήνας (Latitude: 37.9 - 38.1, Longitude: 23.6 - 23.8).
*   **Ανάλυση Διαδρομών:** Υπολογισμός της συνολικής απόστασης που διένυσε ένα όχημα χρησιμοποιώντας τον τύπο Haversine.
*   **Ιστορικό Θέσεων:** Ανάκτηση ιστορικών δεδομένων κίνησης για συγκεκριμένα χρονικά διαστήματα.
*   **Προσομοίωση Δεδομένων:** Περιλαμβάνει ένα εργαλείο παραγωγής δεδομένων (`DataGenerator`) για τη δοκιμή του συστήματος με εικονικά οχήματα.

## Αρχιτεκτονική Συστήματος

Το έργο ακολουθεί μια δομή πολλαπλών επιπέδων (Layered Architecture):

1.  **VehicleTracking.Core:** Περιέχει τις επιχειρησιακές οντότητες (Entities), τα interfaces και την κύρια επιχειρησιακή λογική (`GpsService`).
2.  **VehicleTracking.Data:** Υπεύθυνο για την πρόσβαση στα δεδομένα χρησιμοποιώντας Entity Framework 6 και το Repository Pattern.
3.  **VehicleTracking.Web:** Το επίπεδο παρουσίασης (ASP.NET Web API 2) που εκθέτει τις λειτουργίες στο διαδίκτυο.
4.  **VehicleTracking.DataGenerator:** CLI εφαρμογή για την παραγωγή εικονικής κίνησης οχημάτων.

## Τεχνολογικό Stack

*   **Framework:** .NET Framework 4.6
*   **Web API:** ASP.NET Web API 2
*   **ORM:** Entity Framework 6
*   **Dependency Injection:** Unity Container
*   **Database:** SQL Server (μέσω LocalDB για ανάπτυξη)
*   **JSON Serialization:** Newtonsoft.Json

## Εγκατάσταση και Χρήση

### Προαπαιτούμενα
*   Visual Studio 2022 ή MSBuild
*   SQL Server LocalDB

### Μεταγλώττιση (Build)
Για τη μεταγλώττιση του project μέσω γραμμής εντολών:
```bash
msbuild VehicleTrackingSystem.sln /t:Build /p:Configuration=Debug
```

### Ρύθμιση Βάσης Δεδομένων
Το σύστημα χρησιμοποιεί `DropCreateDatabaseIfModelChanges` ή `VehicleTrackingInitializer` για την αυτόματη δημιουργία και αρχικοποίηση της βάσης δεδομένων με δοκιμαστικά δεδομένα κατά την πρώτη εκτέλεση.

## API Endpoints (Συνοπτικά)

*   `POST /api/gps/position`: Υποβολή θέσης.
*   `POST /api/gps/positions/batch`: Μαζική υποβολή θέσεων.
*   `GET /api/vehicles`: Λίστα όλων των οχημάτων.
*   `GET /api/gps/vehicle/{id}/route`: Ιστορικό διαδρομής και συνολική απόσταση.
